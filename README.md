# itskw.dev — Personal Portfolio

My personal portfolio site built with Next.js 16, featuring 3D visualizations, scroll-driven animations, and a dark glass aesthetic.

**Live:** [itskw.dev](https://itskw.dev)

## Tech Stack

- **Framework:** Next.js 16 / React 19 / TypeScript
- **3D:** Three.js / React Three Fiber
- **Animation:** GSAP / ScrollTrigger / Lenis
- **Styling:** Tailwind CSS 4

## Features

- Liquid text hover effects with elastic easing
- Interactive 3D keyboard visualization
- Horizontal-scroll project showcase
- Smooth scroll with Lenis + GSAP integration
- Dynamic OG image generation
- Responsive across all devices

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
